# Best Practices

- Never send sensitive information in plain text.
- Use consistent branding across all departments.
- Include a clear subject line for every email.
- Avoid large attachments when possible; use secure download links.
- Ensure all automated emails are generated from `noreply@cashtro.in`.
- Maintain professional, concise, and user-friendly communication in every email.
